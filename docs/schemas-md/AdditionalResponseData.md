
# AdditionalResponseData

| *description*:   | *Additional Response Data.*|
|----|----|
| issuingBankName |    ``` string ```  <br/>  *maxLength: 18 example: USA Bank* <br/>  Issuing Bank Name.|
| countryOfIssuance | ``` string ```  <br/>  *maxLength: 3 example: USA*  <br/> Country of Issuance.|
| cardProductID | ``` string ```  <br/>  *maxLength: 1 example: H* <br/>  Card Product ID.|  
| detailedProductID | ``` string ```  <br/>  *maxLength: 3 example: C* <br/>  Detailed Product ID.|
| associationResponseCodeAdtl | ``` string ```  <br/>  *maxLength: 3 example: 00* <br/>  Association Response Code.| 
| cardBrand | ``` string ```  *maxLength: 1 example: M* Card Brand.|
| deferredAuthTransactionID | ``` string ```  <br/>  *maxLength: 16 example: 3561032059730219* <br/>  Deferred Auth Transaction ID for re-authorized (Optimized) transaction|
