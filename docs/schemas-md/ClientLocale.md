
# ClientLocale

| *description*:   | *Country and language of client.*|
|----|----|
| language |     ``` string ```  <br/> *pattern: ^(?!\s*$).+    <br/> *example: en*. Language used by client.|
| country |    ``` string ```  <br/> pattern: ^(?!\s*$).+  <br/> *example: GB*.  <br/> Country of the client.|  

**ClientLocale Example:**

```{r}

{
  "language": "en",
  "country": "GB"
}
```
