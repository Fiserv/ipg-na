
# ClientLocale

| *description*:   | *Country and language of client.*|
|----|----|
| language |     ``` string ``` *pattern: ^(?!\s*$).+   example: en*. Language used by client.|
| country |    ``` string ```  *pattern: ^(?!\s*$).+ example: GB*. Country of the client.|  

**ClientLocale Example:**

```{r}

{
  "language": "en",
  "country": "GB"
}
```
