
# IdInfo

| *description*: | *Identification information.*|
|----|----|
| idType |    ``` string ```   *maxLength: 2 example: DL* Identification type.|
| idData | ``` string ```  *minLength: 1 maxLength: 35 example: 12345678* Identification data. |

**IdInfo Example:**

```{r}

{
  "idType": "DL",
  "idData": "12345678"
}
```
